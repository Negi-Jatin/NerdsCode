10. Write a Java program to create an interface Encryptable with methods encrypt (String data) and decrypt (String encryptedData) that define encryption and decryption operations. Create two classes AES and RSA that implement the Encryptable interface and provide their own encryption and decryption algorithms.

Code:

// Interface for encryptable objects
interface Encryptable {
    String encrypt(String data);
    String decrypt(String encryptedData);
}
                                        
// AES class implementing Encryptable interface
class AES implements Encryptable {
    
    public String encrypt(String data) {
        // Your AES encryption logic goes here
        System.out.println("AES encryption for: " + data);
        return "Encrypted_AES_" + data;
    }
                                        
    
    public String decrypt(String encryptedData) {
        // Your AES decryption logic goes here
        System.out.println("AES decryption for: " + encryptedData);
        return encryptedData.replace("Encrypted_AES_", "");
    }
}
                                        
// RSA class implementing Encryptable interface
class RSA implements Encryptable {
    
    public String encrypt(String data) {
        // Your RSA encryption logic goes here
        System.out.println("RSA encryption for: " + data);
        return "Encrypted_RSA_" + data;
    }
                                        
    
    public String decrypt(String encryptedData) {
        // Your RSA decryption logic goes here
        System.out.println("RSA decryption for: " + encryptedData);
        return encryptedData.replace("Encrypted_RSA_", "");
    }
}
                                        
// Main class
public class EncryptableExample {
    public static void main(String[] args) {
        // Create instances of encryptable objects
        Encryptable aesEncryptor = new AES();
        Encryptable rsaEncryptor = new RSA();
                                        
        // Encrypt and decrypt using AES
        String dataAES = "Sensitive data";
        String encryptedAES = aesEncryptor.encrypt(dataAES);
        String decryptedAES = aesEncryptor.decrypt(encryptedAES);
                                        
        System.out.println("Original Data: " + dataAES);
        System.out.println("Encrypted Data (AES): " + encryptedAES);
        System.out.println("Decrypted Data (AES): " + decryptedAES);
                                        
        System.out.println();
                                        
        // Encrypt and decrypt using RSA
        String dataRSA = "Confidential information";
        String encryptedRSA = rsaEncryptor.encrypt(dataRSA);
        String decryptedRSA = rsaEncryptor.decrypt(encryptedRSA);
                                        
        System.out.println("Original Data: " + dataRSA);
        System.out.println("Encrypted Data (RSA): " + encryptedRSA);
        System.out.println("Decrypted Data (RSA): " + decryptedRSA);
    }
}

Output:

AES encryption for: Sensitive data
AES decryption for: Encrypted_AES_Sensitive data
Original Data: Sensitive data
Encrypted Data (AES): Encrypted_AES_Sensitive data
Decrypted Data (AES): Sensitive data

RSA encryption for: Confidential information
RSA decryption for: Encrypted_RSA_Confidential information
Original Data: Confidential information
Encrypted Data (RSA): Encrypted_RSA_Confidential information
Decrypted Data (RSA): Confidential information
