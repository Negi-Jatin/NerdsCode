9. Write a Java method to find all twin prime numbers less than 100.

Code:

                                public class TwinPrimeFinder {

                                    // Method to check if a number is prime
                                    private static boolean isPrime(int num) {
                                        if (num < 2) {
                                            return false;
                                        }
                                        for (int i = 2; i <= Math.sqrt(num); i++) {
                                            if (num % i == 0) {
                                                return false;
                                            }
                                        }
                                        return true;
                                    }
                                
                                    // Method to find all twin prime numbers less than 100
                                    public static void findTwinPrimes() {
                                        System.out.println("Twin Prime Numbers less than 100:");
                                
                                        for (int i = 3; i < 100; i++) {
                                            // Check if i and i+2 are both prime (twin prime condition)
                                            if (isPrime(i) && isPrime(i + 2)) {
                                                System.out.println("(" + i + ", " + (i + 2) + ")");
                                            }
                                        }
                                    }
                                
                                    public static void main(String[] args) {
                                        // Example usage:
                                        findTwinPrimes();
                                    }
                                }
