10. Write a Java method to count the number of digits in an integer with the value 2. The integer may be assumed to be non-negative.

Code:

public class CountDigitTwo {

    // Method to count the number of occurrences of digit 2 in a non-negative integer
    public static int countDigitTwo(int number) {
        // Convert the integer to a string for easier digit manipulation
        String numberStr = String.valueOf(number);
                                
        // Initialize the count of digit 2
        int count = 0;
                                
        // Iterate through each character in the string
        for (char digitChar : numberStr.toCharArray()) {
            // Check if the character is '2'
            if (digitChar == '2') {
                count++;
            }
        }
                                
        return count;
    }
                                
    public static void main(String[] args) {
        // Example usage:
        int num1 = 12345;
        int num2 = 22222;
                                
        System.out.println("Number of occurrences of digit 2 in " + num1 + ": " + countDigitTwo(num1));
        System.out.println("Number of occurrences of digit 2 in " + num2 + ": " + countDigitTwo(num2));
    }
}

Output:

Number of occurrences of digit 2 in 12345: 1
Number of occurrences of digit 2 in 22222: 5
