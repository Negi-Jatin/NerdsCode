5. Program to append the string using StringBuffer class.

Code:

public class AppendStringBuffer {
public static void main(String[] args) {
    // Create a StringBuffer object
    StringBuffer stringBuffer = new StringBuffer("Hello, ");
                            
    // Append a string to the StringBuffer
    stringBuffer.append("World!");
                            
    // Display the result
   System.out.println("Appended String: " + stringBuffer.toString());
}
}

Output:

Appended String: Hello, World!
